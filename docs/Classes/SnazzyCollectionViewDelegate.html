SnazzyCollectionViewDelegate

public class SnazzyCollectionViewDelegate: NSObject, UICollectionViewDelegate, UICollectionViewDelegateFlowLayout

An instance of SnazzyCollectionViewDelegate is responsable for responding the methods that the collection view needs in order to display all the cells properly. You must have this instance in your desired UIViewController.

    Parameters

    dataSource

    The SnazzyCollectionViewDataSource that you instantiated in your UIViewController.
    reachLastCellInCollection

    An optional callback that will be called when the user reachs the last cell in the UICollectionView.
    reachLastSectionInCollection

    An optional callback that will be called when the user reachs the last section in the UICollectionView.
    didEndScrollingAtIndex

    An optional callback that will be called when the UICollectionView stops scrolling.
    didScroll

    An optional callback what will be called when the UICollectionView scrolls.
    willDisplayCell

    An optional callback that will be called for every cell when it will be displayed.
    willBeganScrolling

    An optional callback that will be called when the UICollectionView began scrolling.

  • cellIsGoingOutWithProgress(offsetY:cell:)

    This method will return the offset of the cell specified in the filter parameter.

    Declaration

    Swift

    public func cellIsGoingOutWithProgress(offsetY additionalOffsetY: CGFloat = 0.0, cell filter: @escaping (SnazzyCollectionCellConfigurator) -> Bool) -> CGFloat?

    Parameters

    offsetY

    You must provide the offsetY of the UICollectionView regarding to its superview. For example if the UICollectionView has a topAnchor to its superview of 30 points, then you should pass the 30 in the offsetY parameter. Otherwise if the UICollectionView has a topAnchor equal its superview, then you don’t need to pass any value in this parameter.
    cell

    Closure that indicates what cell this method should track its progress.

    Return Value

    The progress of the cell from the origin of the UICollectionView.
