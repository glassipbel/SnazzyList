SnazzyTableFooterProtocol

public protocol SnazzyTableFooterProtocol

A Protocol that must be implemented in all your footers.

  • tableView(_:viewForFooterInSection:with:)

    This method will be called whenever the UITableView will call the viewForFooterInSection. You will receive in the parameter item all the values that you entered when you created the configFile.
